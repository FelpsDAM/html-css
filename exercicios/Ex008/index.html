Treinando Negrito e Itálico

Negrito / Destaque

Este é um exemplo de frase escrita em Negrito utilizando a tag <b> (não é semântica pois não tem significado, apenas forma)

Esta outra frase, porém, tem a maneira correta de escrever em Negrito utilizando a tag <Strong>

Itálico / Ênfase

Este parágrafo contém um exemplo de texto em Itálico usando a tag <i> que não possui semântica

O jeito correto de escrever em Itálico é utilizando a tag EM (Emphasis) que é semântica!

Escrevendo texto marcado

É possível escrever um texto com cor de fundo como se estivesse sido marcado por um marca-texto, utilizando a tag MARK

Texto riscado no centro (chamado excluído)

Para escrever um texto riscado no centro, utilize a tag <del>

Texto sublinhado (também chamado de texto inserido)

Para colocar uma linha sublinhando um texto, utilize a tag <ins>
